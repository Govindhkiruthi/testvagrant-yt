Import this project as "File -> Import -> Maven -> Existing Maven Project".
Under <rootfolder>\src/Test/Resource - open the testng.xml file and run it as TestNG -> TestSuite
Under <rootfolder>\test-output folder emailable-report.html  can be found. launch it in browser to view results


Note:

src\test\java 
	contains all the test files, base class files, any utils, constants, etc.,
src\test\resources
    contains all the resources - input.json, testng.xml
Pom.xml - contains the dependencies.
